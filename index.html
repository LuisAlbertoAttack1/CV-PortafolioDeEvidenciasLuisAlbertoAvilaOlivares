El modelo OSI (Open Systems Interconnection) es un marco conceptual que estandariza las funciones de una red de telecomunicaciones o sistema de computación en siete capas: física, de enlace de datos, de red, de transporte, de sesión, de presentación y de aplicación.
Cada capa tiene funciones específicas y se comunica con las capas adyacentes, facilitando la interoperabilidad y la comunicación entre diferentes sistemas y protocolos de red.
Este modelo es fundamental para entender y diseñar redes eficientes y escalables.

¿Qué es una computadra?

Es un dispositivo electrónico que procesa datos mediante programas para realizar diversas tareas, desde cálculos simples hasta operaciones complejas, facilitando la gestión de información y la automatización de procesos.

¿Qué es un maus?

Es un dispositivo de entrada periférico utilizado para interactuar con una computadora, permitiendo al usuario mover un cursor en la pantalla y seleccionar, arrastrar y manipular objetos mediante botones y una superficie de desplazamiento.
